

# Shahid Patel — Professional Engineering Portfolio

A high-end, authoritative portfolio website for a Civil, Structural, Safety & Quality Engineer. No backend required — this is a purely frontend, content-rich static site.

---

## Visual Identity & Theme

- **Color palette**: Charcoal (#1F2933) primary, Steel Blue (#2F4F6F) secondary, Muted Safety Orange accent (sparingly), Off-white/Light Concrete Grey background
- **Typography**: Inter or IBM Plex Sans for headings; clean sans-serif body; monospace-style for stats/tools
- **Style**: Clean grids, architectural spacing, subtle dividers, sharp corners — no rounded bubbly UI. Engineering-drawing-inspired iconography.
- **Tone**: Write like an engineer documenting work. No emotional adjectives. Clarity, structure, precision.
